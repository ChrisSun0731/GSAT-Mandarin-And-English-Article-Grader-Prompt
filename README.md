# GSAT-English-Article-Grader-Prompt
GSAT English Article Grader Prompt

An AI-powered evaluation prompt designed to simulate an **official Taiwan GSAT English Writing grader**. This prompt evaluates students' English essays according to the official scoring standards and provides detailed feedback, corrections, and a high-scoring revised version.

---

## 📌 Overview

The **GSAT English Writing Grader Prompt** helps students, teachers, and educators evaluate English writing performance based on the four official scoring categories:

1. **Content**
2. **Organization**
3. **Grammar & Sentence Structure**
4. **Vocabulary & Spelling**

The AI acts as an official GSAT English Writing examiner and provides:

- A score out of 20
- Category-by-category justification
- Complete grammar and vocabulary error analysis
- Sentence-level corrections
- A revised 20/20 model essay
- Improvement suggestions for future writing tasks

# 📊 Scoring System

The grader follows the official GSAT 20-point writing framework.

| Category | Maximum Score |
|---|---:|
| Content | 5 |
| Organization | 5 |
| Grammar & Sentence Structure | 5 |
| Vocabulary & Spelling | 5 |
| **Total** | **20** |

---

# 📝 Evaluation Criteria

## 1. Content (5 points)

Evaluates:

- Whether the main idea is clear
- Whether the essay answers the prompt
- Whether supporting details are relevant and well-developed

### Score Guidelines

| Score | Description |
|---|---|
| 5–4 | Clear topic with complete and detailed support |
| 3 | Topic is present but insufficiently developed |
| 2–1 | Topic unclear or details mostly irrelevant |
| 0 | Off-topic or blank |

---

## 2. Organization (5 points)

Evaluates:

- Introduction, body, and conclusion structure
- Logical flow
- Use of transitions

### Score Guidelines

| Score | Description |
|---|---|
| 5–4 | Clear structure and effective transitions |
| 3 | Some organization problems |
| 2–1 | Weak coherence and unclear structure |
| 0 | No meaningful organization |

---

## 3. Grammar & Sentence Structure (5 points)

Evaluates:

- Grammar accuracy
- Sentence variety
- Sentence construction
- Punctuation

### Score Guidelines

| Score | Description |
|---|---|
| 5–4 | Few errors, varied sentence structures |
| 3 | Some errors but meaning remains clear |
| 2–1 | Frequent errors affecting readability |
| 0 | Meaning is unclear due to severe errors |

---

## 4. Vocabulary & Spelling (5 points)

Evaluates:

- Word choice
- Vocabulary range
- Spelling
- Capitalization

### Score Guidelines

| Score | Description |
|---|---|
| 5–4 | Accurate and appropriate vocabulary |
| 3 | Simple vocabulary or occasional misuse |
| 2–1 | Frequent vocabulary/spelling problems |
| 0 | Only isolated words or copied phrases |

A table containing:

|-|-|-|-|

| Organization | X | 5 | Explanation |
| Grammar & Sentence Structure | X | 5 | Explanation |

---

## B. Detailed Feedback

The grader explains:

- Why each score was assigned
- Strengths of the essay
- Areas requiring improvement

---

## C. Grammar, Vocabulary, and Spelling Corrections

Every language error is identified.

Format:

| Original | Corrected | Error Type | Explanation |
|-|-|-|-|
| He go to school | He goes to school | Grammar | Third-person singular requires -s |

The grader checks:

- Grammar
- Sentence structure
- Word choice
- Spelling
- Capitalization
- Punctuation

---

## D. Revised Essay (20/20 Version)

- Preserving the student's original ideas

- Creating a GSAT high-scoring response

# ⚠️ Limitations

This AI grader is designed to simulate GSAT evaluation but does not replace official scoring.

Scores may differ from actual GSAT graders because:

- Human graders may interpret ideas differently
- Creativity and style can influence judgment
- Real exam scoring involves professional moderation

Use the feedback as a learning tool rather than an official result.

This project follows these principles:
